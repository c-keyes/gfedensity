# gfedensity

## Authors
Christopher Keyes, Andrew Kobin

## Description
Code repository for computations related to the paper "On p-adic solubility of Ax^l + By^m + Cz^n = 0" by Christopher Keyes and Andrew Kobin

## Files

* gfedensity.sage: sage code file containing routines for computing the p-adic density of (A,B,C) for which Ax^l + By^m + Cz^n = 0 has a primitive p-adic solution
* Example python notebooks (see section 6 of the paper)
  * Example_diag_plane_curves.ipynb: computations related to exponents (l,m,n) = (n,n,n) (see section 6.1 of the paper)
  * Example_22n.ipynb: computations related to exponents (l,m,n) = (2,2,n) (see section 6.2 of the paper)
  * Example_332.ipynb: computations related to exponents (l,m,n) = (3,3,2) (see section 6.3 of the paper)
  * Example_6_10_15.ipynb: computations related to exponents (l,m,n) = (6,10,15) (see section 6.4 of the paper)
  * Example_numerics.ipynb: estimating numerically the density of integers (A,B,C) for which Ax^l + By^m + Cz^n = 0 is everywhere locally soluble when (l,m,n) are pairwise coprime (see section 6.5 of the paper)
